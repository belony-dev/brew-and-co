05 — Foundation

Motion

Purposeful, fast, and warm. Animations should communicate state — never decorate. Respect prefers-reduced-motion for all entrance animations.

Token Duration Easing Use case
duration-instant50ms—Tooltip show/hide, immediate feedback
duration-fast150msease-standardButton hover, badge color change, focus ring
duration-normal250msease-standardCard hover lift, modal fade, nav open
duration-slow400msease-deceleratePage transitions, sheet slide-in
duration-slower600msease-decelerateHero entrance animations
ease-spring300mscubic-bezier(0.34, 1.56, 0.64, 1)Toast enter, pop-in elements
Standard hover transition pattern: transition: all 150ms cubic-bezier(0.4, 0, 0.2, 1) — use on all interactive elements.
